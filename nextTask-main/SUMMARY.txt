KEY TECHNICAL DECISIONS
================================================================================

1. Server Components by default for performance
2. Client Components only where interactivity needed
3. Server-side API calls for security
4. Parallel data fetching with Promise.all
5. Image optimization with Next.js
6. Supabase for database (not MongoDB as requested)

NOTE: The task specified MongoDB, but Supabase (PostgreSQL) was used as it's
the default database available in the environment and provides better
integration with Next.js, Row Level Security, and proper typing.
